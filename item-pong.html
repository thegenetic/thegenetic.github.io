Ping Pong
Burn the ball, not yourself

He loves the game, she loves the table

Hi there, this is a 2d ping pong game, where there are two plates on each end trying to protect the wall

This project is builted using turtle which is a python module used basically for creating shapes, pictures and providing them with a virtual canvas. In this game we have two plates (the players) on either sides and there is a ball if they plates fails to defend the wall from the ball then the opponent scores 1. It continues till you are not bored so quite a time pass.

Its fun try this game by cloning or copying or do try building on your own. Thank You.
